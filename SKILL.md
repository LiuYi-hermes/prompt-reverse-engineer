# Prompt Reverse Engineer

## Overview

将任意图片反推为完整的文本提示词。用 **7 大板块 / 72 个参数** 的结构化体系分析图片，再转换为各平台最优的提示词格式。

> **铁律**：画面上的水印/文字/Logo等后期叠加元素不属于画面内容，**禁止反推进提示词**。反推的是照片本身，不是照片上的修图标签。

> **维护注意**：本技能的核心竞争力在于输出模板的还原度。增强时优先在现有模板上做像素级细节扩展（如面部精度、材质描述），而非替换整体输出结构。v4.0曾因引入四轮递进/证据锚点/交叉验证等机制导致输出冗长失真，已回退。保持简洁精确。

## Core Workflow

```
输入图片 → 五层递进分析 → 参数体系填充 → 平台适配输出 → 本地存档
```

### Step 1: 接收输入

支持任意来源的图片：用户上传、URL链接、本地文件路径。

### Step 2: 五层递进分析

按 **由外到内、由粗到细** 的顺序逐层观察：

1. **画面框架 A** — 先看整体：这是什么类型（照片/插画/3D/截图）？画幅比例？景别？场景？色彩基调？情绪？空间层次？

2. **人物主体 B** — 有人则细看：
   - **比例与位置**：人物占画面高度百分比？头顶距画面上边缘的百分比？脚底距下边缘的百分比？人物宽度占画面宽度的百分比？面部占人物高度的比例？面部占画面总面积的百分比？
   - **姿势逐关节**（从头顶到脚尖）：头部朝向（左/右旋转角度、前倾/后仰角度）、颈部扭转方向与角度、肩膀（左右高度差、是否耸肩、肩线与水平夹角）、双臂（大臂与躯干夹角、小臂与大臂夹角、肘部在身体的精确高度位置、手腕位置、每根手指的弯曲/伸展/抓握状态）、躯干（与垂直方向的前倾/后仰角度、左右侧弯方向与角度）、髋部（左右高低差、是否前顶/后坐）、双腿（大腿与垂直方向夹角、膝盖在画面中的精确水平%和垂直%位置、小腿与大腿夹角、脚踝状态、脚掌着地方式、脚尖指向方向）
   - **面部与表情**（像素级精度）：
     - **整体结构**：脸型精确形状（鹅蛋/瓜子/方圆/心形），脸宽脸长比例，下颌线形状，三庭五眼比例，额头高度与饱满度，颧骨位置与突出程度，面部对称性
     - **眉眼区域**：眉形（柳眉/剑眉/平眉/弯眉），眉峰弧度，眉毛长度与眼宽比例，密度与毛流感，眉间距精确距离，眉色与发色协调度；眼型（杏眼/丹凤眼/圆眼/桃花眼/下垂眼），眼距占脸宽比例，眼裂长度，眼睑厚度，上眼睑弧度，瞳孔颜色，虹膜纹理与反光点，睫毛长度/密度/是否根根分明，眼白纯净度与血管纹理，双眼皮类型（内双/外双/单眼皮），神态（温柔/魅惑/凌厉/空洞/专注），眼神方向（直视镜头/凝视画面外某处）
     - **鼻部**：鼻梁高度（高挺/中等/低平），鼻根位置，鼻翼宽度占脸宽比例，鼻头形状（圆润/尖翘/肉感），鼻孔可见度，鼻梁侧面线条，鼻基底高度
     - **口唇区域**：唇形（樱桃小嘴/饱满厚唇/薄唇），上唇弧度和厚度，下唇饱满度，唇峰唇谷细节，唇色精确色值（自然粉/豆沙/正红/裸色），唇纹理可见度，牙齿排列与颜色（若露齿），口角位置，微表情（上扬/闭合/微张/咬唇）
      - **皮肤质感**：真实肤色色值（暖白/象牙白/小麦色），毛孔可见度与分布，细小纹理，皮下血管隐现，subsurface scattering效果，皮脂光泽（哑光/轻微光泽/水光），雀斑/痣/疤痕位置与尺寸，细纹分布区域
   - **表情逐肌肉**：眉毛内侧/外侧上扬或下沉、眼轮匝肌收缩状态、上眼睑覆盖虹膜比例、下眼睑是否收紧、鼻翼是否扩张、口轮匝肌收缩状态、嘴角上扬精确角度、上下唇间距、露齿数量、苹果肌隆起程度、下巴收紧或前伸状态
   - **发饰逐件**：头发上所有装饰物——发夹/发圈/发带/发簪/头箍/发绳，每件描述颜色、材质、佩戴位置、尺寸
   - **饰品逐件**（从头顶到脚尖全列）：眼镜（镜框形状/颜色/材质/粗细/镜片反光/镜腿设计）、项链/项圈（材质/颜色/宽度/长度/吊坠形状尺寸颜色材质）、耳饰（长度/造型/垂落位置）、乳环/脐环/体环（颜色/直径/粗细/装饰）、手镯/手链/戒指（每只手每件的材质/数量/佩戴手指位置）、脚链/脚环、纹身（文字内容/字体风格/颜色/相对于骨骼的精确位置/尺寸）、包（形状/尺寸/颜色/材质/提手类型/五金颜色）
   - **妆造逐层**：底妆遮瑕区域与高光修容分区、眉妆工具痕迹（眉笔/眉粉/纹绣）和眉头眉峰眉尾虚实变化、眼妆（眼线起止位置和粗细变化/眼影颜色和晕染范围/睫毛真假类型）、唇妆（唇线笔痕迹/是否涂满/咬唇渐变/唇蜜叠加）、腮红（颜色/位置/范围直径/与修容融合程度）
    - **服装与层次**：服装类型、颜色、材质、纹样？层次结构（内衣/外衣/披风/腰带）？褶皱与垂坠状态？
   - **身形与肤色**：身高比例？体型特征？曲线分布？肤色色调与质感？

3. **摄影语言 C** — 镜头怎么拍：焦距/光圈/镜片？角度？构图？景深？透视？清晰度？
4. **光影系统 D** — 光从哪里来：主光/补光/轮廓光/环境光？光比？特殊光效？
5. **后期风格 E** — 什么质感：调色？对比/饱和度？滤镜/胶片/CCD？材质质量？
6. **输出载体 G** — 这是什么载体：纯照片还是模拟了杂志/手机截图/角色设定表/海报？

### Step 3: 参数体系填充

使用完整参数体系（详见 `references/parameter_system.md`）标注图片特征。

**每个参数只填可观测到的特征**，不确定的标记为"推测"或留空。

核心参数🔴为必填，增强参数🟡为建议填，可选参数🟢按需填。

### Step 4: 输出提示词

**默认目标平台：GPT Image 2.0，中文自然语言。**

#### GPT Image 2.0 中文版（唯一输出格式）

```
【格式要求】
- 中文自然语言，建议500-1500字（根据图片复杂度）
- 结构：Scene（场景/画幅）→ 比例与位置 → 姿势逐关节 → 表情逐肌肉 → 面部五官（整体/眉眼/鼻/唇/皮肤）→ 发饰 → 饰品逐件 → 妆造逐层 → 服装道具 → 光影氛围 → 后期风格 → 技术强化

【整体面部结构】
脸型精确：{脸型，脸宽与脸长比例约1:X.XX}，面部对称性极高，三庭五眼比例严格符合原始数据，额头高度适中饱满，颧骨位置自然不突出

【眉眼区域】
眉毛：{眉形}，眉峰弧度精确，眉毛长度占眼宽的约X倍，密度适中，眉间距精确为{距离}，眉色与发色协调
眼睛：{眼型}，眼距适中（眼间距占脸宽约1/X），眼裂长度精确，眼睑厚度自然，上眼睑弧度流畅，瞳孔颜色{具体}，虹膜纹理细腻可见反光点，睫毛长度{描述}、密度高且根根分明，眼白纯净带有自然血管纹理，神态{描述}，完全匹配原始眼神

[重要细节——材质质感、颜色关系、飘带/特效的动态轨迹、空间关系]。

【光影与氛围】
光照条件：{光源类型与方向}，主光{软硬/色温}，辅以{补光/轮廓光}描述，精确还原原始照片的光影分布，高光位置在鼻梁、额头、唇部等{具体位置}，反光点自然，阴影柔和不夸张，整体氛围{描述，例如：干净摄影棚光 / 自然窗光 / 夜景街灯}
```

**注意**：仅输出此格式。不输出英文版、不输出其他平台（Midjourney/DALL-E/Stable Diffusion/Seedream）。
