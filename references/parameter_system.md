# 反推图片提示词 - 参数体系设计 v3.0（完美复刻级·最终版）

## 设计原则
1. **能复刻**：任一 AI 人像图，按此参数填满即可在 Seedream 5.0 / GPT Image 2.0 上重现
2. **可操作**：每个枚举值都是 AI 能识别的具体描述
3. **分层执行**：🔴核心（每图必反推）→ 🟡增强（可见则推）→ 🟢可选（精度优化）
4. **不冗余**：父子层级清晰，一个细节只出现一次
